## 摘要
混合专家（MoE）通过选择性地将输入路由到专门的专家子集来有效扩展模型容量。然而，输入-专家专业化（MoE的核心动机）关键取决于路由器是否真正意识到输入结构。在实践中，MoE路由通常实现为浅层线性投影，对输入表示的感知有限，这常常导致路由不稳定。我们提出STAR，一种结构感知路由，它将MoE路由重新思考为子空间学习问题，通过用演化主子空间增强标准可学习路由，该子空间通过广义Hebbian算法（GHA）跟踪主导输入结构。通过将路由决策直接与输入结构对齐，STAR实现了稳定的专家专业化。我们在受控合成设置以及大规模语言和视觉任务上评估了STAR，它始终在强MoE基线上改进了路由质量和下游性能。此外，可选的测试时子空间更新进一步增强了输入分布偏移下的路由鲁棒性和泛化能力。

## Abstract
Mixture-of-Experts (MoE) scales model capacity efficiently by selectively routing inputs to a specialized subset of experts. However, input-expert specialization, the core motivation of MoE, critically depends on whether the router is actually aware of input structure. In practice, MoE routing is typically implemented as a shallow linear projection with limited awareness of input representation, which often leads to unstable routing. We propose STAR, a Structure Aware Routing that rethinks MoE routing as a subspace learning problem by augmenting standard learnable routing with an evolving principal subspace that tracks dominant input structure via Generalized Hebbian Algorithm (GHA). By aligning routing decisions directly with input structure, STAR enables stable expert specialization. We evaluate STAR on controlled synthetic setup and large-scale language and vision tasks, where it consistently improves routing quality and downstream performance over strong MoE baselines. Moreover, optional test-time subspace updates further enhance routing robustness and generalization under input distribution shifts.

### 论文详细总结：STAR: Rethinking MoE Routing as Structure-Aware Subspace Learning

#### 1. 核心问题与整体含义（研究动机和背景）
- **问题**：混合专家（MoE）模型通过稀疏门控将输入路由到子集专家，从而高效扩展模型容量。然而，其核心路由机制通常是浅层线性投影 + Softmax，对输入数据的**结构（分布变化）** 感知有限，导致路由不稳定、专家专业化不充分、负载不均。
- **背景**：现有工作主要聚焦于负载均衡（如辅助损失），但未能从根本上确保路由器“意识到”输入变化。论文提出：路由应视为**子空间学习**问题，使决策直接对齐输入的主成分结构。

#### 2. 方法论：核心思想、关键技术细节、公式与算法流程
- **核心思想**：在标准可学习线性路由器（Gating Matrix）基础上，**增量式学习一个低维主子空间**（通过广义Hebbian算法 GHA），并将两个路由分数进行插值，使路由兼顾任务监督信号与输入数据自身结构。
- **关键技术细节**：
  1. **主子空间构建**：使用 GHA 在线估计输入隐层表示的**前 K 个主成分**（右奇异向量），得到正交基矩阵 \(V \in \mathbb{R}^{K \times d}\)。GHA 更新公式（Sanger 规则）：
     \[
     v_i \leftarrow v_i + \eta\left(y_i x - \sum_{j=1}^{i} y_j v_j\right), \quad y_i = v_i^\top x
     \]
     每次前向传播执行 \(m\) 次迭代（通常 \(m=1\) 或 3），可在线逼近 PCA 结果。
  2. **基混合矩阵 \(R\)**：在 GHA 基上引入可学习的 \(R \in \mathbb{R}^{K \times K}\)，得到路由向量 \(Z = RV\)。作用：解耦专家选择与主成分的方差排序，避免能量集中到前几个主成分。理论分析（Lemma 4.1, Proposition 4.2, 4.3）证明无混合（\(R=I\)）时路由能量等于特征值，导致不平衡；随机正交混合可均化期望能量；学习式 \(R\) 在实验中有效避免坍缩。
  3. **双路由插值**：
     \[
     s = \mathrm{Softmax}(\sigma(\alpha) \odot l_{\text{linear}} + (1 - \sigma(\alpha)) \odot l_{\text{GHA}})
     \]
     其中 \(l_{\text{linear}} = xW_g^\top\)（标准线性门控），\(l_{\text{GHA}} = xZ^\top\)（结构感知门控），\(\alpha \in \mathbb{R}^K\) 经 Sigmoid 后作为逐专家插值系数。
  4. **算法流程**（Algorithm 1）：初始化 \(W_g, V, R, \alpha\) 及专家权重；每输入 \(x\)，先执行 \(m\) 次 GHA 更新 \(V\)，再计算两个 logits 并插值、Softmax，最后用 top-k 选专家。
  5. **测试时自适应（TTA）**：推理时可继续启用 GHA 更新，使子空间适应测试分布偏移，无需调整其他参数。
