## 摘要
混合专家多模态大语言模型（MoE-MLLMs）性能卓越，但GPU内存成本过高，因此压缩不可或缺。在PTQ方法中，专家级混合精度量化已被证明对MoE-LLMs有效，但由于专家重要性估计中两个被忽视的偏差，其在MoE-MLLMs上性能显著下降。（1）在跨模态层面，视觉标记的数量优势导致专家选择频率由视觉标记主导，掩盖了对文本模态至关重要的专家；（2）在视觉内部层面，大量冗余视觉标记进一步扭曲频率统计，模糊了对信息性视觉内容关键专家的识别。为弥补这些差距，我们提出MODE，一种面向MoE-MLLMs的模态分解专家级混合精度量化框架，它按模态分解专家选择频率，过滤冗余视觉标记以获得去噪视觉频率，并进一步评估每模态的量化敏感性作为基于频率估计的补充信号。这些信号被整合到整数线性规划公式中，以在给定预算下分配每个专家的位宽。大量实验表明，MODE特别适合MoE-MLLMs，在W3A16设置下将平均性能损失限制在2.9%以内，在极端2位设置下获得更大增益。

## Abstract
Mixture-of-Experts Multimodal Large Language Models (MoE-MLLMs) offer remarkable performance but incur prohibitive GPU memory costs, making compression essential. Among PTQ methods, expert-level mixed-precision quantization has proven effective for MoE-LLMs, yet suffers notable degradation on MoE-MLLMs due to two overlooked biases in expert importance estimation. (1) At the cross-modal level, the numerical dominance of vision tokens causes expert selection frequency to be dominated by vision tokens, masking experts that are critical to the text modality; (2) at the intra-vision level, the large proportion of redundant vision tokens further skew frequency statistics, obscuring experts critical for informative visual content. To bridge gaps, we propose MODE, a modality-decomposed expert-level mixed-precision quantization framework for MoE-MLLMs that decomposes expert selection frequency by modality, filters redundant vision tokens to obtain denoised visual frequency, and further evaluates quantization sensitivity per modality as a complementary signal to frequency-based estimation. These signals are integrated into an Integer Linear Programming formulation to assign per-expert bit-widths under a given budget. Extensive experiments show that MODE is particularly well-suited for MoE-MLLMs, limiting average performance loss to within 2.9% at W3A16, with larger gains at the extreme 2-bit setting.

### 1. 论文的核心问题与整体含义（研究动机和背景）

*   **核心问题**：混合专家多模态大语言模型（MoE-MLLMs）虽然性能出色，但其庞大的参数规模（如Qwen3-VL-30B-A3B-Instruct）导致惊人的 GPU 内存消耗（62GB），严重阻碍了其在资源受限场景下的部署。现有的后训练量化（PTQ）方法，如针对密集MLLMs或单模态MoE-LLMs设计的量化方法，在应用到MoE-MLLMs时效果不佳。
*   **研究动机**：作者识别出两个被现有专家级混合精度量化方法忽略的关键偏差，这些偏差导致了专家重要性估计不准确，进而损害了量化后的模型性能：
    1.  **跨模态偏差 (Cross-Modal Bias)**：在多模态输入中，视觉 token 的数量（例如 360个）远多于文本 token（例如 15个）。因此，基于全局token计算出的专家选择频率会被视觉 token 主导，从而严重低估了对文本推理至关重要的专家。
    2.  **视觉内偏差 (Intra-Vision Bias)**：视觉 token 中存在大量冗余信息。这些冗余 token 与关键 token 会系统地激活不同的专家群体。由于冗余 token 数量占优，使用全部视觉 token 计算的频率会高估那些被冗余 token 偏好的专家，而低估了对真正有信息量的视觉内容至关重要的专家。
*   **整体含义**：论文旨在解决 MoE-MLLMs 量化中专家重要性估计不准的痛点，通过提出一种模态感知的混合精度量化框架，实现更精准的位宽分配，从而在极低比特（如W3A16）下实现高性能的模型压缩与部署。

### 2. 论文提出的方法论：核心思想、关键技术细节

*   **核心思想**：**MODE** 的核心是“模态分解”（Modality-Decomposed）。它将专家的重要性评估分解为文本和视觉两个维度，并进一步在视觉维度内筛选出关键信息，从而获得更精确、无偏的专家重要性信号，最终通过整数线性规划求解最优的每位专家位宽分配。
*   **关键技术细节（三步走）**：
    1.  **第一步：模态感知的专家选择频率统计**
        *   不再使用所有token的全局频率。
        *   分别收集**文本 token** 和**关键视觉 token** 的专家选择频率。
        *   **关键视觉 token** 的定义：在每一层，根据该层注意力机制中所有文本 token 对视觉 token 的总注意力权重，选取排名前20%的视觉 token 作为关键视觉 token（用于捕捉核心视觉语义）。
        *   对每种模态的频率进行层内归一化，得到 `f_t(e)` 和 `f_v(e)`。
    2.  **第二步：模态感知的量化敏感性评估**
        *   频率反映专家被调用的次数，但敏感性反映量化造成的实际损害。
        *   对每个专家 `e` 和候选位宽 `b`，分别计算仅将该专家量化为 `b` 比特后，**仅对文本 token 输出** 产生的损失（KL散度）`δ_T(e, b)`，以及**仅对关键视觉 token 输出** 产生的损失 `δ_V(e, b)`。
        *   这实现了专家重要性和量化损失的解耦。
    3.  **第三步：基于整数线性规划（ILP）的位宽分配**
        *   将位宽分配形式化为一个优化问题。
        *   **优化目标**：最小化所有专家的模态加权损失之和，即 `Σ_e [ f_t(e)*δ_T(e,b) + f_v(e)*δ_V(e,b) ]`。
        *   **约束条件**：总位宽等于预设的平均位宽目标（如3比特或2比特），且每个专家只能分配一个位宽。
        *   通过求解 ILP 来获得每个专家的最优位宽分配方案。

### 6. 论文的主要结论与发现

1.  现有的专家级混合精度量化方法在 MoE-MLLMs 上失效的根本原因是跨模态和视觉内在的两个专家重要性估计偏差。
2.  将专家选择频率按文本和（关键）视觉模态分离，可以显著纠正偏差，提升重要性估计的准确性。
3.  模态分解的量化敏感性是频率信号的有效补充，两者结合能更全面地指导位宽分配。
4.  **MODE 在所有实验设置下（W3A16, W2A16）均显著优于所有对比方法**。例如，在 Qwen3-VL-30B-A3B 模型上，W3A16 设置下平均精度仅下降 2.84%，W2A16 设置下比最强基线高出 4% 以上。
5.  MODE 与旋转式量化技术（如QuaRot）正交，可以结合以获得进一步性能提升。
6.  通过3-bit量化，一个30B规模的MoE-MLLM可以部署在单个消费级GPU（如RTX 4090）上，同时保持较高的精度（平均70.01%）。
