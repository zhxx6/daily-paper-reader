## 摘要
训练后量化（PTQ）对于大语言模型（LLM）的高效部署至关重要。最近超低位宽PTQ方法依赖于刚性的权重显著性假设或位置启发式，引入了大量隐藏的缩放开销。我们提出SAGE-PTQ（显著性感知图引导高效PTQ），一种新颖的LLM超低位宽量化框架，能够最小化隐藏缩放成本。SAGE-PTQ使用分布统计分离显著和非显著权重，然后将子采样的非显著权重建模为稀疏图，以估计每层最优分组数。SAGE-PTQ采用双模量化，对显著权重分配多比特精度，对非显著权重进行二值化。为减少缩放开销，SAGE-PTQ对显著权重使用每通道一个缩放因子，对每个非显著分组使用一个标量缩放因子。最后，SAGE-PTQ实现自适应显著性阈值选择每个矩阵的最优显著性比率。SAGE-PTQ平均每个矩阵仅需1.03权重比特和0.004缩放比特，优于BiLLM和PB-LLM等最先进方法。在LLaMA-3-8B上，SAGE-PTQ实现6.74的WikiText2困惑度，而BiLLM为55.8，同时使用不到BiLLM 50%的GPU内存。在LLaMA-2-70B上，SAGE-PTQ在单张NVIDIA L40 GPU上提供1.5倍解码加速，展示了实际推理效率。

## Abstract
Post-training quantization (PTQ) is critical for the efficient deployment of large language models (LLMs). Recent ultra-low-bit PTQ methods rely on rigid weight-saliency assumptions or position heuristics, introducing substantial hidden scaling overhead. We propose SAGE-PTQ (Saliency-Aware Graph-guided Efficient PTQ), a novel ultra-low-bit quantization framework for LLMs that minimizes hidden scaling cost. SAGE-PTQ separates salient and unsalient weights using distributional statistics, then models subsampled unsalient weights as a sparse graph to estimate the optimal number of groups per layer. SAGE-PTQ applies dual-mode quantization, assigning multi-bit precision to salient weights and binarizing unsalient weights. To reduce scaling overhead, SAGE-PTQ uses one per-channel scale for salient weights and one scalar per unsalient group. Finally, SAGE-PTQ implements adaptive saliency thresholding to select the optimal saliency ratio per matrix. SAGE-PTQ achieves 1.03 weight bits and only 0.004 scaling bits per matrix on average, outperforming state-of-the-art methods such as BiLLM and PB-LLM. On LLaMA-3-8B, SAGE-PTQ achieves 6.74 WikiText2 perplexity, compared to 55.8 for BiLLM, while using less than 50% of BiLLM's GPU memory. On LLaMA-2-70B, SAGE-PTQ provides 1.5x faster decoding on one NVIDIA L40 GPU, demonstrating practical inference efficiency.

  3. **子优量化**：简单min-max缩放导致大量量化误差，且基于块的缩放引入隐藏开销（超过1比特/权重）。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：将权重矩阵分为显著（salient）和非显著（unsalient）两部分，对显著权重采用多比特量化（如4比特），对非显著权重采用二值化（实际为三元 {-1,0,1}），并通过图引导算法自适应确定非显著权重的最优分组数，同时采用自适应显著性阈值优化每矩阵的显著性比例，最小化量化误差和存储开销。

- **关键技术细节**：
  1. **显著性感知权重划分**：
     - 基于幅度（magnitude）的显著性度量（优于Hessian度量），根据权重分布统计（均值β和标准差γ）通过分位数函数（probit）确定阈值，分离显著子集Ω_sal和非显著子集Ω_uns。
  2. **图引导非显著分组估计**：
     - 对非显著权重进行随机子采样（采样率0.03%），构建稀疏K近邻（KNN）图。
     - 应用谱聚类（spectral clustering），使用轮廓系数（Silhouette Score）评估不同候选分组数（r ∈ [2^(NLUT-1)+1, 2^NLUT-1]），选择最优分组数N_opt_uns。
     - 基于分位数将非显著权重划分为N_opt_uns个不相交的子集。
  3. **双模量化（Dual-Mode Quantization）**：
     - **模式1（显著量化）**：使用单一通道级缩放因子（per-channel scale）和N_b比特的离散矩阵，通过交替优化求解缩放因子和量化值，最后映射到量化级别。
     - **模式2（非显著二值化）**：每个非显著分组使用一个标量缩放因子，通过元素级阈值规则得到三元矩阵{-1,0,1}，并闭式求解最优标量。
     - 最终整体二值矩阵由所有非显著分组的矩阵和显著矩阵的符号层叠加得到。
  4. **自适应显著性阈值选择**：
     - 将显著性比例ζ_sal视为优化变量，目标为最小化归一化重构误差（Frobenius范数），约束在[0, ζ_sal_max]之间。使用Brent方法求解最优阈值ζ_opt_sal。
  5. **高效推理运行时**：
     - 设计位打包查找表（lookup table, LUT）编码每组索引，大小不超过NLUT比特，存储于主机内存，按层加载，避免昂贵的解码开销。

- **公式/算法流程**（文字说明）：
  - 整体目标：min_{W_hat} ||W - W_hat||_F^2，其中W_hat = A B，A为对角缩放矩阵，B为量化矩阵。
  - 显著性滤波：Ω_sal = {w ∈ Ω | |w| > β + γ·Φ^{-1}(ζ_sal/2)}。
  - 图引导分组：通过稀疏KNN图 + 谱聚类 + 轮廓分数搜索最优组数。
  - 双模量化：显著部分交替优化缩放和离散值；非显著部分直接阈值+闭式解。
  - 自适应阈值：Brent方法最小化重构误差。（详细算法见论文Algorithm 1和Algorithm 2）
