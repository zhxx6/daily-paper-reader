## 摘要
训练后量化（PTQ）是一种广泛采用的技术，用于在不重新训练的情况下压缩大型语言模型（LLM）。现有的二阶PTQ方法，包括GPTQ，仅从输入激活统计量构建量化目标，实质上假设所有输出通道对逐层重建目标的贡献相等。我们提出KronQ，一种通过将梯度协方差引入量化流程来挑战这一假设的PTQ框架。在Kronecker分解海森近似下，量化损失同时依赖于激活协方差和梯度协方差，KronQ在两个互补层面上利用这一点。（1）KronQ引入了双向非相干处理，将现有的输入侧随机旋转扩展到输出维度，利用梯度协方差，从而减少输入和输出维度上的权重幅度方差。（2）KronQ推导出一种新的灵敏度度量，用于层间混合精度分配，该度量由梯度和激活海森迹驱动。值得注意的是，在LLaMA-3-70B的2比特仅权重量化情况下，尽管GPTQ和GPTAQ产生发散或退化的量化结果（在WikiText-2上困惑度超过2000），KronQ实现了7.93的困惑度。

## Abstract
Post-training quantization (PTQ) is a widely adopted technique for compressing large language models (LLMs) without retraining. Existing second-order PTQ methods, including GPTQ, construct quantization objectives exclusively from input activation statistics, effectively assuming that all output channels contribute equally to the layer-wise reconstruction objective. We propose KronQ, a PTQ framework that challenges this assumption by introducing the gradient covariance into the quantization pipeline. Under the Kronecker-factored Hessian approximation, the quantization loss depends jointly on both the activation and gradient covariances, and KronQ exploits this at two complementary levels. (1) KronQ introduces bidirectional incoherence processing, extending the existing input-side random rotation to the output dimension using the gradient covariance, reducing weight magnitude variance across both input and output dimensions. (2) KronQ derives a new sensitivity metric for inter-layer mixed-precision allocation, driven by the gradient and activation Hessian traces. Notably, in the case of 2-bit weight-only quantization on LLaMA-3-70B, while GPTQ and GPTAQ diverge or produce degenerate quantizations (>2000 perplexity on WikiText-2), KronQ achieves 7.93 perplexity.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题背景**：大型语言模型（LLMs）参数规模达数百亿，部署时内存和计算开销巨大。后训练量化（PTQ）是无需重新训练的有效压缩方法，其中补偿型方法（如GPTQ）基于最优脑外科医生（OBS）原理，通过输入激活协方差矩阵 \(H_X\) 作为代理海森矩阵来补偿权重量化误差。
- **核心缺陷**：现有方法仅依赖输入侧二阶信息 \(H_X\)，隐含假设所有输出通道对重建损失贡献相等（即 \(H_G = I\)）。但实际中，输出通道的梯度幅度差异可达数个数量级（如图1a所示），忽略梯度协方差 \(H_G\) 导致在极低比特（2-bit）下量化极易发散或退化。
- **整体意义**：KronQ首次将梯度协方差有效引入LLM后训练量化流程，通过Kronecker因子海森近似更准确地建模量化损失，显著提升了极低比特量化的稳定性和精度。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：基于Kronecker因子海森近似 \(H \approx H_X \otimes H_G\)，将完整二阶信息纳入量化目标。\(H_G = \mathbb{E}[gg^\top]\) 通过一次反向传播从校准集获得，存储为 \(d_{\text{out}} \times d_{\text{out}}\) 矩阵。
- **关键技术细节**：
  - **双向非相干处理（Bidirectional Incoherence Processing, BiIP）**：
    - 先进行输入侧和输出侧对角缩放：\(W \leftarrow S_G W S_X\)，其中 \(S_X, S_G\) 分别由 \(H_X\) 和 \(H_G\) 的对角项与权重范数导出。
    - 再应用随机正交变换（随机化Hadamard变换）使得 \(H_X\) 和 \(H_G\) 都变得非相干，降低权重幅度变异系数（如图2b所示）。变换后量化目标保持不变（定理1）。
  - **权重量化更新**（命题1）：在K-FAC近似下，梯度协方差 \(H_G\) 在列式OBS更新中代数抵消，补偿公式简化为与GPTQ相同的形式，额外只需GPTAQ的非对称校正项，因此KronQ继承GPTAQ的计算效率，无需修改量化求解器。
  - **混合精度分配**：提出联合海森迹灵敏度 \(s_\ell = \operatorname{tr}(H_G^{(\ell)}) \cdot \operatorname{tr}(H_X^{(\ell)})\)。该指标能区分共享相同输入的Q/K/V投影层（它们 \(H_X\) 相同但 \(H_G\) 不同），而激活仅 \(H_X\) 指标则无法区分。分配高比特给灵敏度大的子层。
- **算法流程**（算法1和2）：
  1. 预计算 \(H_G\)（一次性反向传播）。
  2. BiIP预处理：缩放+随机正交变换。
  3. 按GPTAQ方式进行逐列量化，利用 \(H_X\) 的Cholesky因子和GPTAQ校正矩阵。
  4. 量化完成后逆变换回原始空间。

## 6. 主要结论与发现

- KronQ在所有设置与比特下一致优于或持平现有方法，最大增益在2-bit。例如：
  - LLaMA-3-70B W2：GPTQ和GPTAQ发散（PPL>2000），KronQ达到7.93。
  - LLaMA-2-7B W2A4：GPTAQ为10.91，KronQ降至9.38。
- 双向非相干处理有效降低权重行列变异系数，且HG代数抵消无需增加量化求解器复杂度。
- 混合精度灵敏度\( \operatorname{tr}(H_G)\cdot\operatorname{tr}(H_X) \)提供比纯激活迹更优的位宽分配，可免费从已计算的HG获得。

## 7. 优点

1. **创新性**：首次在补偿型PTQ中系统利用输出侧梯度协方差，打破了传统仅依赖激活统计的瓶颈。
2. **高效性**：HG在OBS更新中抵消，无需修改量化循环；一次反向传播即可获得，且可在BiIP后释放内存。
3. **有效性**：在极低比特（2-bit）下效果显著，解决了GPTQ等发散问题；混合精度分配无需搜索或额外代价。
4. **通用性**：在多种模型、量化方式、新模型和困难基准上验证，跨家族有效。
5. **理论严谨**：提供旋转不变性定理（定理1）和LDLQ最优性上界（命题2）。

## 8. 不足与局限

1. **额外的离线开销**：需要一次完整反向传播来估计\( H_G \)，增加校准时间（约+27%每层），且可能增加内存临时峰值（表8）。
2. **存储与内存**：\( H_G \)为全矩阵，对宽层（如隐藏维4096）约64MB，在70B级模型上累计较大。虽然有释放机制，但校准过程内存仍高于GPTAQ。
3. **推理开销**：双向Hadamard旋转增加\( O(d_{\text{in}}\log d_{\text{in}} + d_{\text{out}}\log d_{\text{out}}) \)计算，虽可通过CUDA融合减轻但仍高于无旋转方法。
4. **适用范围限制**：仅针对线性层，对其他类型层（如卷积、Embedding）未验证；未测试更先进的矢量量化（QuIP#的E8P码本）组合。
5. **偏差风险**：校准集仅用128条WikiText-2，可能对领域外数据不够鲁棒；梯度协方差估计依赖校准集，若校准分布与测试分布偏差大，性能可能下降。
6. **消融实验**：仅展示了W2下的消融，未在更高比特下验证各组件贡献。
