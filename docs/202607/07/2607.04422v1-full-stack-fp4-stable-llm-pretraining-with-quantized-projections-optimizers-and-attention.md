## 摘要
最近的NVFP4预训练方法主要针对Transformer线性层，而优化器状态、优化器算术和注意力机制在4比特流水线中的研究不足。这一关键缺口阻碍了稳定的全栈4比特预训练，因为这三个核心模块表现出独特的数值失败模式：线性层受到维度传播误差放大导致的硬量化噪声限制；AdamW二阶矩是重尾非负值，对低精度分母脆弱；注意力机制携带易出错的计算路径，要求严格的前向-后向量化一致性。我们提出了全栈FP4，这是第一个完整的NVFP4预训练框架，通过模块级精度策略解决了所有三个稳定性瓶颈。对于线性投影，LoRA-SVD轻量级分解抑制了量化噪声，打破了直接量化误差天花板，将仅线性层的损失差距从1.40%缩小到0.61%。对于优化器，我们设计了AdamW二阶矩变换以实现稳健的NVFP4存储，并完全支持Root (Muon)优化器的原生NVFP4 Newton-Schulz迭代。对于注意力机制，采用混合精度方案量化Q/K/V和反向dS，同时保护BF16中的脆弱路径，并配以统一张量复用维持前向-后向对齐。我们进一步分析了朴素低比特矩阵乘法中快速误差累积以及PV / dOV^T注意力分支的极端敏感性。所有模块即插即用，具有累积稳定性和效率提升。我们的3B/64B-token预训练验证了近BF16性能，仅有1.47%的损失差距，验证了可行的稳定端到端NVFP4大语言模型预训练。

## Abstract
Recent NVFP4 pretraining methods mainly target transformer linear layers, leaving optimizer states, optimizer arithmetic and attention underexplored in 4-bit pipelines. This critical gap blocks stable full-stack 4-bit pretraining, as the three core modules exhibit unique numerical failure patterns: linear layers hit hard quantization noise limits with dimension-propagated error amplification; AdamW second moments are heavy-tailed non-negative values fragile to low-precision denominators; attention carries error-prone computation paths demanding strict forward-backward quantization consistency. We propose Full-Stack FP4, the first complete NVFP4 pretraining framework resolving all three stability bottlenecks via module-wise precision strategies. For linear projections, LoRA-SVD lightweight decomposition suppresses quantization noise and breaks the direct-quantization error ceiling, shrinking the linear-only loss gap from 1.40% to 0.61%. For optimizers, we design AdamW second-moment transformation for robust NVFP4 storage and fully support native NVFP4 Newton-Schulz iterations for the Root (Muon) optimizer. For attention, a mixed-precision scheme quantizes Q/K/V and backward dS while guarding vulnerable paths in BF16, paired with unified tensor reuse to sustain forward-backward alignment. We further analyze fast error accumulation in naive low-bit matrix multiplication and the extreme sensitivity of PV / dOV^T attention branches. All modules are plug-and-play with cumulative stability and efficiency improvements. Our 3B/64B-token pretraining validates near-BF16 performance with merely 1.47% loss gap, verifying feasible stable end-to-end NVFP4 LLM pretraining.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：当前大语言模型（LLM）预训练的计算成本推动精度从FP32降至BF16和FP8，Blackwell GPU已原生支持NVFP4（4-bit），使得4-bit预训练从理论走向实践。然而，现有4-bit预训练方法主要集中在Transformer的线性层（权重/激活/梯度），而**优化器状态、优化器计算（如Muon/Root的Newton-Schulz迭代）以及注意力机制这三个核心模块**在4-bit精度下的稳定性问题尚未得到系统解决。这三个模块各自存在独特的数值失败模式：
  - **线性投影**：直接NVFP4量化存在固有的量化噪声天花板，且误差随隐藏维度线性增长（附录A证明），最终淹没优化信号。
  - **优化器**：AdamW的二阶矩 `v_t` 是非负、重尾分布的，且作为分母对量化敏感；Root(Muon)的Newton-Schulz迭代在低精度下数值脆弱。
  - **注意力**：包含多个高敏感路径（如 `P V` 和 `dOV^T`），且要求前向-后向量化一致性，否则会导致发散。
- **整体含义**：本文旨在证明**稳定的端到端全栈NVFP4 LLM预训练是可行的**，并提出了首个涵盖线性投影、优化器（状态与计算）和注意力的完整方案，使得所有模块均可在NVFP4精度下协同工作。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **整体架构**：采用模块化精度策略，针对每个瓶颈设计专门方案，所有模块可即插即用（见图1）。
  - **线性投影 → LoRA-SVD**：
    - 核心思想：将权重视为“主成分+残差”，通过一个低秩BF16分支（LoRA）保留主成分，残差部分用NVFP4计算，从而稀释量化噪声，打破直接4-bit量化的误差天花板。
    - 关键技术：
      - **子空间约束的反向传播**：使用快速Cholesky-QR分解确保低秩分支始终捕获主成分。前向：`Y = NVFP4_GEMM(Q(X), Q(W_res)) + β * (X L1^T) L2^T`；反向类似，但避免重复计算大矩阵乘法。
      - **周期性随机SVD重新对齐**：每4k步用随机SVD更新低秩分支，应对子空间漂移。
      - 误差分析表明，LoRA-SVD使输出、输入梯度、权重重度的相对MSE降低约10%（图2）。
  - **优化器**：
    - **Quantized-State AdamW (Q-AdamW)**：针对二阶矩 `v_t`，设计4步变换管道以匹配NVFP4的存储范围：
      1. 平方根平滑：`ṽ = √v`；
      2. 块内均值减除：`ṽ = ṽ - μ`（按tile计算均值）；
      3. 快速Hadamard变换（16×16）；
      4. NVFP4量化（得到缩放因子和量化值）。重建时逆操作。
      - 效果：使二阶矩量化误差降低约60%（图3a），否则训练直接失败。
      - 一阶动量 `m_t` 使用Hadamard旋转+随机舍入+4/6自适应缩放量化。
    - **NVFP4 Root (Muon) 优化器 (Q-Root)**：使Newton-Schulz迭代仅用NVFP4原生计算，通过形状优化的系数（`a, b, c`）保持正交性和稳定性，避免混合精度辅助运算。窄形状参数回退到Q-AdamW。
      - 量化NS迭代流程（公式略），使用RTN和4/6缩放。优化系数可有效降低误差（图3b）。
  - **注意力 → 混合精度FP4注意力 (Q-Attn)**：
    - 设计混合精度方案：**Q/K/V和反向dS使用NVFP4**，而**`P V`、`dO V^T`、`P^T dO`（即Softmax及其相关路径）使用BF16**，同时保证前向和后向使用相同的量化Q/K/V张量（一致性）。
    - 理论分析（附录D）解释为何 `P V` 和 `dOV^T` 路径敏感：Softmax的稀疏性导致块级量化破坏概率单纯形约束（`P V`），或梯度噪声被概率峰值放大（`dOV^T`）。
